# <%=className%> Web Component

## Install
```bash
npm i -s @quincarter-components/<%=fileName%>
```
or with `yarn`
```bash
yarn add @quincarter-components/<%=fileName%>
```

## Usage
```html
<<%=fileName%>></<%=fileName%>>
```

### Development
* Update package.json with scripts needed for testing or additional functions
